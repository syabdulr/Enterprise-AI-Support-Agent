## 🐛 Troubleshooting

### "python: command not found"
Run this command:
```bash
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 1
```

### Permission denied
Make the demo executable:
```bash
chmod +x demo/simple_demo.py
```

### Module not found
The simple demo doesn't require dependencies, but if you see errors:
```bash
cd /home/openclaw/enterprise-ai-support-agent
python demo/simple_demo.py
```
